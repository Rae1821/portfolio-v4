Fitness Pro Workout Tracker

Overview

Fitness Pro is a workout tracker with full CRUD abilities. It features a minimal, clean design with an easy to use form that lets you create, update, and delete your workouts. On your personalized dashboard you can see all of the workouts you have created and sort them by type. At the top you can see the number of workouts you completed for the current year, the current and previous months, and the total number of workouts you have completed.

Objective

I wanted an easy simple app that I could use to keep track of my workouts. My workouts vary and I did not have one central place to keep track of them all. I was using the notes app on my phone but that was getting long and tedious so I decided to use my tech skills to create the workout tracker I was missing.

Role: Web designer + developer

  • UI design
  • Front end development

Timeline: six weeks

Deliverables: Mobile responsive website with CRUD capabilities

Tools: Nextjs, TailwindCSS, MongoDB, Git/GitHub, Vercel, NextAuth

Result + Value Provided

In creating the workout tracker I decided to incorporate different types of workouts one might complete based on my friends who all workout in different ways. That way the website would be beneficial to multiple users and not just those into strength training.
